빠른 이해를 돕기 위해, 상황에 따라 `fn main() { ..`을 생략합니다. 예를 들어,

```rust
fn main() {
    println!("Hello, world!");
}
```

는

```rust
println!("Hello, world!");
```

으로 표기합니다.
